description: "Interactive demos of all built-in placement slot components with live theme switching."
---

The Component Gallery renders each built-in placement slot directly in the browser — no Sandpack or iframe needed. Switch between story variants and theme presets to see how each component responds.

## Available Components

| Component | Type | Description |
|-----------|------|-------------|
| [ButtonSlot](/components/button-slot/) | `button` | Standalone CTA button for nav bars and inline prompts |
| [BannerSlot](/components/banner-slot/) | `banner` | Full-width banner for announcements and usage warnings |
| [ModalSlot](/components/modal-slot/) | `modal` | Overlay modal for feature gates and trial expirations |
| [ToastSlot](/components/toast-slot/) | `toast` | Ephemeral toast notification for usage alerts |
| [QuotaMeterSlot](/components/quota-meter-slot/) | `in_page` | Usage meter with progress bar, counter, and gauge styles |
| [InlineEmbedSlot](/components/inline-embed-slot/) | `in_page` | Inline card for feature gates and upgrade prompts |

Each page includes:
- **Variant selector** — switch between story variants (e.g. Primary, Secondary, Accent)
- **Theme selector** — Default, Dark, and Warm Brand presets
- **Live rendering** — components render with the SDK's real theme system
